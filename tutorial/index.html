You've just created a simple Play Framework application with the Slick library and Bootstrap3!
This template will help you building a variety of Web applications.
Also you will be able to easily execute the Slick code generator.

View the App

Once the application has been compiled and the server started, your application can be accessed at: http://localhost:9000

Slick code generator

You can execute the Slick code generator from top page.It generate source code from DB schema.
In order to enable source code generation you can specify DB schema you want to have in the application.conf file.
The default value is : 

  db.default.url="jdbc:h2:mem:play"

For example, When you specify 'INFOMATION_SCHEMA' to a shema on the top page, you can generate the source code based on the schema.
